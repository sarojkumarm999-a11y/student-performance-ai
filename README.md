# 🎓 Student Performance Prediction AI

NLP + LLM + Machine Learning system to predict student GPA, Grade, and Pass/Fail.

## Tech Stack
- Python, Flask, Scikit-learn, Pandas
- Anthropic Claude API
- HTML, CSS, JavaScript

## Features
- Predicts GPA (0-4.0)
- Predicts Grade (A/B+/B/C/D/F)
- Pass/Fail classification
- NLP behavioral analysis
- Personalized AI insights

## How to Run
```bash
pip install -r requirements.txt
python main.py
```
Then open http://localhost:5000
